# w3-d3-fullstack-inventory-manager
Objective Build a small full stack feature with React and Flask.

This app generates a list of randomized items one might find in a dungeon crawler game, and when the user hits a button to explore/roll for loot, it will randomly assign that item and it's rarity and add it to the user's inventory. This can be repeated until reaching max total weight. 

The user can also trash items to empty the inventory to start over. 

To run the frontend, CLI npm run dev from the inventory-manager folder

To run the backend, CLI py app.py from backend folder or
py backend/app.py
